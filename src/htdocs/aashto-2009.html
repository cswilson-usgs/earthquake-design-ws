Request Parameters (Input)

latitude
Latitude of location of interest, in decimal degrees
Example: 34.05
longitude
Longitude for location of interest, in decimal degrees
Example: -118.25
siteClass
Site Class, as defined in Table 3.4.2.1-1 of seismic design reference document
Options: A for "Hard rock", B for "Rock", C for "Very dense soil and soft rock", D for "Stiff soil", A (See Table 3.4.2.1-1)
title
Title for location/request
Example: Example

Output

Web service response

request
date
Timestamp
Example: “YYYY-MM-DDThh:mm:ssZ”
referenceDocument
Identifying the reference document for the returned data
Example: “AASHTO-2009”
status
String ("success" or "error")
Example: “success”
url
URL to reproduce this result
Example: “{MOUNT_PATH}/aashto-2009.json?latitude=34&longitude=-118&riskCategory=III&siteClass=C&title=Example”
parameters
latitude
Latitude for location at which to compute composite results
Example: 34
longitude
Longitude for location at which to compute composite results
Example: -118
riskCategory
Risk category for structure
Example: “III”
siteClass
NEHRP site classification
Example: “C”
title
Name for site/report
Example: “Example”
response
data
pga
PGA, the mapped horizontal Peak Ground Acceleration, in units of g
fpga
FPGA, the site coefficient for PGA, from Table 3.4.2.3-1
as
As = FPGA x PGA (Equation 3.4.1-1), the design PGA
ss
MCER ground motion (0.2 s)
fa
Site amplification factor (0.2 s)
sds
Numeric seismic design value (0.2 s)
s1
The mapped 1-second spectral acceleration, in units of g
fv
Site amplification factor (1.0 s)
sd1
Numeric seismic design value (1.0 s)
sdc
Seismic design category from SDCS and SDC1
ts
TS = SD1 / SDS (Equation 3.4.1-6), in seconds, for construction of the design response spectrum
t0
T0 = 0.2TS (Equation 3.4.1-5), in seconds, for construction of the design response spectrum
sdSpectrum
Design horizontal response spectrum
metadata
griddedValuesID
Identifier for the underlying gridded values
spatialInterpolationMethod
Identifier for the interpolation method used to obtain data for location of interest from underlying gridded datasets
